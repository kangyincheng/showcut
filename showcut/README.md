# ShowCut - 截图工具

一款功能强大的 Windows 截图工具，支持窗口自动对齐、图形标注、翻译等功能。

## 功能特性

### 📷 截图功能
- 全屏区域选择截图
- 窗口自动对齐（鼠标悬停自动识别窗口边界）
- 实时显示截图尺寸
- 支持快捷键 Ctrl+Shift+A

### ✏️ 标注工具
- **矩形工具** - 绘制矩形框
- **圆形工具** - 绘制圆形/椭圆
- **直线工具** - 绘制直线
- **箭头工具** - 绘制带箭头的线
- **多边形工具** - 绘制多边形（点击添加顶点，双击或按Enter完成）

### 🎨 线条样式
- 颜色选择器 + 6种预设颜色（红、黄、蓝、绿、黑、白）
- 线条粗细可调（1-20像素）
- 多种线型：实线、虚线、点线、点划线

### 🌐 翻译功能
- 支持多种语言互译（中、英、日、韩、法、德、西）
- 自动检测源语言
- 免费在线翻译接口

### 💾 其他功能
- 保存为 PNG/JPG/BMP 格式
- 一键复制到剪贴板
- 撤销标注
- 清除所有标注

## 快速开始

### 方式一：直接运行源码

1. 安装 Python 3.8 或更高版本
2. 安装依赖：
   ```
   pip install -r requirements.txt
   ```
3. 运行程序：
   ```
   python main.py
   ```

### 方式二：打包为 EXE（免安装）

在 Windows 系统下运行：
```
build.bat
```

打包完成后，可执行文件位于 `dist/ShowCut.exe`，直接双击即可运行，无需安装。

## 使用说明

### 开始截图
1. 点击「开始截图」按钮，或按快捷键 `Ctrl+Shift+A`
2. 鼠标悬停在窗口上会自动识别窗口边界（蓝色高亮）
3. 点击窗口自动对齐选中该窗口区域
4. 或拖拽鼠标手动选择截图区域
5. 按 `Enter` 或双击确认截图，按 `Esc` 取消

### 添加标注
1. 截图完成后，在工具栏选择绘图工具
2. 在截图上拖拽绘制图形
3. 多边形工具：点击添加顶点，双击或按 `Enter` 完成
4. 按 `Esc` 取消当前绘制
5. 按 `Ctrl+Z` 撤销上一个标注

### 调整样式
- 点击颜色按钮或预设颜色块更改线条颜色
- 拖动粗细滑块调整线条宽度
- 从线型下拉框选择实线/虚线/点线/点划线

### 翻译功能
1. 点击「翻译」按钮
2. 在弹出窗口中输入或粘贴文本
3. 选择源语言和目标语言
4. 点击「翻译」按钮获取结果

### 保存与分享
- 「保存」按钮或 `Ctrl+S`：保存为图片文件
- 「复制」按钮或 `Ctrl+C`：复制到剪贴板

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+A | 开始截图 |
| Ctrl+S | 保存截图 |
| Ctrl+C | 复制到剪贴板 |
| Ctrl+Z | 撤销标注 |
| Esc | 取消当前操作 |
| Enter | 确认操作 |
